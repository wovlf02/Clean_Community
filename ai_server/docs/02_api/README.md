# 📡 API 명세

> AI 혐오 표현 탐지 API 엔드포인트 문서

## 📋 개요

| 항목 | 값 |
|------|------|
| **Base URL** | `http://localhost:8000` |
| **프로토콜** | HTTP/HTTPS |
| **Content-Type** | `application/json` |

---

## 🔗 엔드포인트 목록

| 메서드 | 경로 | 설명 | 인증 |
|--------|------|------|------|
| `GET` | `/` | 서버 정보 | ❌ |
| `GET` | `/health` | 상태 확인 | ❌ |
| `POST` | `/analyze` | 단일 텍스트 분석 | ❌ |
| `POST` | `/batch` | 배치 텍스트 분석 | ❌ |

## 🚀 빠른 시작

### 헬스 체크

```bash
curl http://localhost:8000/health
```

### 단일 텍스트 분석

```bash
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{"text": "분석할 텍스트"}'
```

### 배치 분석

```bash
curl -X POST http://localhost:8000/batch \
  -H "Content-Type: application/json" \
  -d '{"texts": ["텍스트1", "텍스트2"]}'
```

---

## 📊 응답 코드

| 코드 | 설명 |
|------|------|
| `200` | 성공 |
| `400` | 잘못된 요청 (유효성 검증 실패) |
| `500` | 서버 내부 오류 |
| `503` | 서비스 이용 불가 (모델 미로드) |

## 🏷️ 지원 라벨

분석 결과에 포함될 수 있는 혐오 카테고리:

| 라벨 | 설명 |
|------|------|
| `여성/가족` | 여성 및 가족 관련 혐오 |
| `남성` | 남성 관련 혐오 |
| `성소수자` | LGBTQ+ 관련 혐오 |
| `인종/국적` | 인종 및 국적 관련 혐오 |
| `연령` | 연령 관련 혐오 |
| `지역` | 특정 지역 관련 혐오 |
| `종교` | 종교 관련 혐오 |
| `기타 혐오` | 기타 유형의 혐오 |
| `악플/욕설` | 일반적인 악성 댓글 |
