### 1.3 컴포넌트 스펙

| 컴포넌트 | 속성 |
|----------|------|
| 로고 | height: 48px |
| 슬로건 | text: h4, color: neutral-600 |
| 이메일 입력 | type: email, 필수 |
| 비밀번호 입력 | type: password, 토글 아이콘 |
| 자동 로그인 | checkbox |
| 로그인 버튼 | Primary, Large, 전체 너비 |
| 소셜 로그인 | Outline, 각 브랜드 컬러 |

### 1.4 인터랙션

| 이벤트 | 동작 |
|--------|------|
| 로그인 클릭 | 유효성 검사 → API 호출 → 성공 시 홈으로 이동 |
| 비밀번호 토글 | 비밀번호 표시/숨김 전환 |
| 소셜 로그인 | OAuth 팝업 또는 리다이렉트 |
| 회원가입 클릭 | `/register`로 이동 |
| 비밀번호 찾기 | `/forgot-password`로 이동 |

### 1.5 에러 상태

```
[이메일/비밀번호 오류]
┌─────────────────────────────────────────────────────┐
│ 📧  user@email.com                                  │
└─────────────────────────────────────────────────────┘
  ⚠️ 이메일 또는 비밀번호가 일치하지 않습니다.
  (color: error, border: error)
```
