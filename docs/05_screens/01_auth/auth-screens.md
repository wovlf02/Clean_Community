**관련 문서**: [디자인 시스템](../00_common/design-system.md) | [기능 요구사항](../../02_requirements/functional.md)

## 화면 목록

인증 도메인의 각 화면은 개별 문서로 분리되어 있습니다.

| 화면 | 파일 | 경로 | 설명 |
|------|------|------|------|
| 로그인 | [login-page.md](./login-page.md) | `/login` | 아이디/비밀번호 로그인 + 소셜 로그인 |
| 회원가입 | [register-page.md](./register-page.md) | `/register` | 4단계 회원가입 (기본정보 → 이메일인증 → 프로필 → 약관) |
| 아이디 찾기 | [find-id-page.md](./find-id-page.md) | `/find-id` | 이메일 인증 후 아이디 확인 |
| 비밀번호 재설정 | [forgot-password-page.md](./forgot-password-page.md) | `/forgot-password` | 이메일 인증 후 비밀번호 재설정 |

## 화면 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  로그인 (LoginPage)                                                         │
│  ├── 아이디/비밀번호 로그인 → 홈                                           │
│  ├── 소셜 로그인 (Google/Kakao/Naver) → OAuth → 홈                         │
│  ├── 아이디 찾기 → 아이디 찾기 (FindIdPage)                                │
│  ├── 비밀번호 찾기 → 비밀번호 재설정 (ForgotPasswordPage)                  │
│  └── 회원가입 → 회원가입 (RegisterPage)                                    │
│                                                                             │
│  회원가입 (RegisterPage)                                                    │
│  ├── Step 1: 기본 정보 (아이디, 비밀번호)                                  │
│  ├── Step 2: 이메일 인증 (6자리 인증번호)                                  │
│  ├── Step 3: 프로필 정보                                                   │
│  └── Step 4: 약관 동의 → 완료 → 로그인                                     │
│                                                                             │
│  아이디 찾기 (FindIdPage)                                                   │
│  ├── Step 1: 이메일 입력 및 인증 요청                                      │
│  ├── Step 2: 6자리 인증번호 확인                                           │
│  └── Step 3: 아이디 확인 → 로그인 또는 비밀번호 재설정                     │
│                                                                             │
│  비밀번호 재설정 (ForgotPasswordPage)                                       │
│  ├── Step 1: 아이디 입력 → 이메일로 인증번호 발송                          │
│  ├── Step 2: 6자리 인증번호 확인                                           │
│  └── Step 3: 새 비밀번호 설정 → 완료 → 로그인                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 공통 규칙

### 레이아웃 구조

- **Desktop**: 가운데 정렬된 카드 레이아웃 (max-width: 420px)
- **Mobile**: 전체 너비 레이아웃 (padding: 24px)

### 접근 권한

| 화면 | 권한 |
|------|------|
| 로그인, 회원가입, 아이디 찾기, 비밀번호 재설정 | 비로그인 사용자만 |

### 소셜 로그인

| 플랫폼 | 아이콘 스타일 |
|--------|---------------|
| Google | 흰색 배경, 원형, 48px |
| Kakao | 노란색 배경 (#FEE500), 원형, 48px |
| Naver | 초록색 배경 (#03C75A), 원형, 48px |

### 이메일 인증

- 이메일은 본인인증 및 아이디/비밀번호 찾기에만 사용
- 인증번호: 6자리 숫자
- 유효시간: 5분
- 재발송 쿨타임: 30초
