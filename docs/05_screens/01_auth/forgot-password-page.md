# 비밀번호 재설정 화면 (ForgotPasswordPage)

**관련 문서**: [디자인 시스템](../00_common/design-system.md) | [기능 요구사항](../../02_requirements/functional.md)

---

## 1. 경로 및 접근

| 항목 | 내용 |
|------|------|
| **경로** | `/forgot-password` |
| **접근 권한** | 비로그인 사용자만 |

---

## 2. 단계별 흐름

```
Step 1: 아이디 입력 → Step 2: 이메일 인증 → Step 3: 비밀번호 재설정 → 완료
```

---
