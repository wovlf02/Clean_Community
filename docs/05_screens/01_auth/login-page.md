## 4. 컴포넌트 스펙

| 컴포넌트 | 속성 |
|----------|------|
| 로고 | height: 48px |
| 슬로건 | text: h4, color: neutral-600 |
| 아이디 입력 | type: text, 필수, placeholder: "아이디를 입력하세요" |
| 비밀번호 입력 | type: password, 토글 아이콘, placeholder: "비밀번호를 입력하세요" |
| 자동 로그인 | checkbox |
| 로그인 버튼 | Primary, Large, 전체 너비 |
| 소셜 로그인 | 원형 아이콘 버튼 (48px), 각 브랜드 컬러 |

---

## 5. 인터랙션

| 이벤트 | 동작 |
|--------|------|
| 로그인 클릭 | 유효성 검사 → API 호출 → 성공 시 홈으로 이동 |
| 비밀번호 토글 | 비밀번호 표시/숨김 전환 |
| 소셜 로그인 아이콘 클릭 | OAuth 팝업 또는 리다이렉트 |
| 회원가입 클릭 | `/register`로 이동 |
| 아이디 찾기 클릭 | `/find-id`로 이동 |
| 비밀번호 찾기 클릭 | `/forgot-password`로 이동 |
