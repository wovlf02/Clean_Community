## 1. 경로 및 접근

| 항목 | 내용 |

| **경로** | `/register` |

## 2. 단계별 흐름

```
Step 1: 기본 정보          Step 2: 이메일 인증        Step 3: 프로필 정보        Step 4: 약관 동의
├─ 아이디 (중복확인)       ├─ 이메일 입력             ├─ 이름                   ├─ 이용약관 (필수)
├─ 비밀번호                ├─ 인증번호 발송           ├─ 닉네임 (중복확인)      ├─ 개인정보 처리방침 (필수)
└─ 비밀번호 확인           └─ 인증번호 6자리 입력     ├─ 프로필 이미지 (선택)   └─ 마케팅 수신 (선택)
                                                      └─ 자기소개 (선택)
```

## 7. 약관 보기 모달

> [보기] 버튼 클릭 시 해당 약관 전체 내용이 표시되는 페이지로 이동

```
[보기] 클릭 시 동작:
• 이용약관 [보기] → /terms 페이지로 새 탭 열기
• 개인정보 처리방침 [보기] → /privacy 페이지로 새 탭 열기
• 마케팅 정보 수신 [보기] → 마케팅 동의 상세 모달 표시
```

## 8. 유효성 검사

| 필드 | 규칙 |
|------|------|
| 아이디 | 4-20자, 영문/숫자, 중복 불가 |
| 비밀번호 | 8자 이상, 영문/숫자/특수문자 포함 |
| 비밀번호 확인 | 비밀번호와 일치 |
| 이메일 | 이메일 형식, 인증 완료 필수 |
| 이름 | 2-20자 |
| 닉네임 | 2-20자, 중복 불가, 특수문자 제한 |
