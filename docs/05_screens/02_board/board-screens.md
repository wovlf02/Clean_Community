# 게시판 화면 설계 (Board Screens)

**관련 문서**: [디자인 시스템](../00_common/design-system.md) | [기능 요구사항](../../02_requirements/functional.md)

---

## 화면 목록

게시판 도메인의 각 화면은 개별 문서로 분리되어 있습니다.

| 화면 | 파일 | 경로 | 설명 |
|------|------|------|------|
| 게시글 목록 | [post-list-page.md](./post-list-page.md) | `/board` | 게시글 목록, 검색, 필터링 |
| 게시글 상세 | [post-detail-page.md](./post-detail-page.md) | `/board/[id]` | 게시글 상세, 댓글, 공유 |
| 게시글 작성/수정 | [post-write-page.md](./post-write-page.md) | `/board/write`, `/board/[id]/edit` | 게시글 작성, 수정, AI 감정분석 |

---

## 화면 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  게시글 목록 (PostListPage)                                                 │
│  ├── 게시글 클릭 → 게시글 상세 (PostDetailPage)                            │
│  └── + 글쓰기 → 게시글 작성 (PostWritePage)                                │
│                                                                             │
│  게시글 상세 (PostDetailPage)                                               │
│  ├── 수정 → 게시글 수정 (PostWritePage)                                    │
│  ├── 삭제 → 삭제 확인 모달 → 목록으로                                     │
│  ├── 공유 → 공유 모달 (카카오톡, 인스타그램, 카페)                         │
│  └── 댓글/대댓글 작성 → AI 감정분석 → 등록                                 │
│                                                                             │
│  게시글 작성 (PostWritePage)                                                │
│  └── 게시 → AI 감정분석 → 경고 시 모달 → 등록/수정                         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 공통 규칙

### AI 감정분석 결과 표시

| 라벨 | 아이콘 | 표시 위치 |
|------|--------|-----------|
| 긍정 | 😊 | 게시글 카드, 게시글 상세, 댓글 |
| 중립 | 😐 | 게시글 카드, 게시글 상세, 댓글 |
| 주의 | 😟 | 게시글 카드, 게시글 상세, 댓글 |
| 경고 | ⚠️ | 게시글 카드, 게시글 상세, 댓글 |

### 공유 플랫폼 (한국)

| 플랫폼 | 아이콘 | 연동 방식 |
|--------|--------|-----------|
| URL 복사 | 🔗 | 클립보드 복사 |
| 카카오톡 | 💬 | Kakao Share API |
| 인스타그램 | 📷 | Instagram Story Share |
| 네이버 카페 | ☕ | 네이버 카페 글쓰기 연동 |

### 접근 권한

| 화면 | 권한 |
|------|------|
| 게시글 목록, 게시글 상세 | 전체 (읽기) |
| 게시글 작성 | 로그인 필수 |
| 게시글 수정/삭제 | 작성자만 |
