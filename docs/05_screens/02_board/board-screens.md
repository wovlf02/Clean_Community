### 3.3 AI 감정분석 경고 (한국어 텍스트 대상)

> **적용 시점**: 사용자가 [게시] 버튼 클릭 → Next.js 서버 → FastAPI 감정분석 → 악성 표현 감지 시 경고 모달 표시
> **적용 대상**: 게시글 제목 + 본문 (한국어 텍스트만)

```
         ┌─────────────────────────────────────────────────────────┐
         │                                                         │
         │                      ⚠️                                 │
         │                                                         │
         │            유해한 표현이 감지되었습니다                  │
         │                                                         │
         │  ───────────────────────────────────────────────────   │
         │                                                         │
         │  작성하신 내용에서 다음과 같은 표현이 감지되었습니다:   │
         │                                                         │
         │    • 🔴 혐오 표현 (신뢰도: 85%)                         │
         │    • 🟠 차별적 표현 (신뢰도: 72%)                       │
         │                                                         │
         │  ───────────────────────────────────────────────────   │
         │                                                         │
         │  💡 건강한 커뮤니티 문화를 위해 표현을 수정해주시면     │
         │     더 좋은 소통이 가능합니다.                          │
         │                                                         │
         │                                                         │
         │      [수정하기]           [그대로 등록]                 │
         │      (Primary)            (Ghost/Gray)                  │
         │                                                         │
         └─────────────────────────────────────────────────────────┘

[동작 설명]
• [수정하기] 클릭: 모달 닫힘, 에디터로 돌아가 내용 수정 가능
  - 수정 후 재게시 시 다시 감정분석 수행
  - 부정적 표현이 줄어들거나 사라지면 정상 등록 처리

• [그대로 등록] 클릭: 현재 내용 그대로 등록 처리
  - 게시글 정상 등록됨
  - 백엔드에서 해당 게시글을 모니터링 대상으로 자동 플래깅
  - 시스템 관리자 대시보드에서 조회/관리 가능

[악성 표현 라벨 종류]
• 🔴 혐오 표현
• 🔴 욕설/비속어
• 🟠 차별적 표현
• 🟠 공격적 표현
• 🟡 선정적 표현
• 🟡 불쾌감 유발
```
