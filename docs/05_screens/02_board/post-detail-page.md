## 3. 감정분석 결과 표시

```
[게시글 상단 - 감정분석 라벨]

┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  [Q&A] · 😊 긍정                                                            │
│                                                                             │
│  게시글 제목이 여기에 표시됩니다                                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

[감정분석 라벨 종류]
• 😊 긍정 - 긍정적인 내용 (bg: green-50, text: green-700)
• 😐 중립 - 중립적인 내용 (bg: gray-100, text: gray-600)
• 😟 주의 - 부정적 표현 감지 (bg: orange-50, text: orange-700)
• ⚠️ 경고 - 유해 표현 감지 (bg: red-50, text: red-700)

[클릭 시 상세 툴팁]
         ┌─────────────────────────────────────┐
         │  AI 감정분석 결과                   │
         │  ─────────────────────────────────  │
         │  분류: 긍정적 표현                  │
         │  신뢰도: 87%                        │
         │  ─────────────────────────────────  │
         │  이 게시글은 긍정적인 표현이        │
         │  감지되었습니다.                    │
         └─────────────────────────────────────┘
```

         │  │   복사  │  │         │  │ 스토리  │  │         │   │

         │                                                         │
         └─────────────────────────────────────────────────────────┘

[공유 옵션 상세]
• URL 복사: 클립보드에 URL 복사, "복사되었습니다" 토스트
• 카카오톡: 카카오톡 공유 API 연동, 미리보기 포함
• 인스타그램 스토리: 인스타그램 스토리 공유 (이미지 + 링크)
• 카페: 네이버 카페 글쓰기 연동
