## 3. 게시글 카드 상세 - 감정분석 결과 표시

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  [게시글 카드]                                                              │
│                                                                             │
│  ┌───────────────────────────────────┐                                     │
│  │                                   │                                     │
│  │        Thumbnail (16:9)           │                                     │
│  │                                   │                                     │
│  └───────────────────────────────────┘                                     │
│                                                                             │
│  [Q&A] · 😊 긍정                      ← 카테고리 + AI 감정분석 결과        │
│                                                                             │
│  게시글 제목이 여기에 표시됩니다                                           │
│  (h3, bold, 2줄 말줄임)                                                    │
│                                                                             │
│  본문 미리보기가 여기에 표시됩니다...                                      │
│  (2줄 말줄임)                                                              │
│                                                                             │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                             │
│  👤 닉네임 · 2시간 전                                                      │
│  👁 123  ❤️ 45  💬 12                                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

[감정분석 결과 라벨]
• 😊 긍정 - 긍정적인 내용 (초록색)
• 😐 중립 - 중립적인 내용 (회색)
• 😟 주의 - 부정적 표현 감지 (주황색)
• ⚠️ 경고 - 유해 표현 감지 (빨간색)
```
