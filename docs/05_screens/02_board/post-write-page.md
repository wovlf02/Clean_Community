````markdown
## 3. AI 감정분석 경고 모달

> **적용 시점**: 사용자가 [게시] 버튼 클릭 → Next.js 서버 → FastAPI 감정분석 → 유해 표현 감지 시 경고 모달 표시
> **적용 대상**: 게시글 제목 + 본문 (한국어 텍스트)

```
         ┌─────────────────────────────────────────────────────────┐
         │                                                         │
         │                      ⚠️                                 │
         │                                                         │
         │            유해한 표현이 감지되었습니다                  │
         │                                                         │
         │  ───────────────────────────────────────────────────   │
         │                                                         │
         │  작성하신 내용에서 다음과 같은 표현이 감지되었습니다:   │

## 4. AI 감정분석 라벨 종류 (모델 라벨링 기준)

> 실제 AI 모델의 라벨링 기준을 그대로 사용

| 라벨 | 아이콘 | 색상 | 설명 |
|------|--------|------|------|
| 여성/가족 | 🔴 | error-500 | 여성, 가족에 대한 비하/혐오 표현 |
| 남성 | 🔴 | error-500 | 남성에 대한 비하/혐오 표현 |
| 성소수자 | 🔴 | error-500 | 성소수자에 대한 비하/혐오 표현 |
| 인종/국적 | 🔴 | error-500 | 인종, 국적에 대한 차별/혐오 표현 |
| 연령 | 🔴 | error-500 | 나이에 대한 차별/비하 표현 |
| 지역 | 🟠 | warning-500 | 지역 비하 표현 |
| 종교 | 🟠 | warning-500 | 종교에 대한 비하/혐오 표현 |
| 기타 혐오 | 🟠 | warning-500 | 기타 혐오/차별 표현 |
| 악플/욕설 | 🔴 | error-500 | 욕설, 비속어, 악성 댓글 |
| clean | ✅ | success-500 | 유해 표현 없음 |

## 5. 동작 설명

```
[수정하기] 클릭 시:
• 모달 닫힘
• 에디터로 돌아가 내용 수정 가능
• 수정 후 재게시 시 다시 감정분석 수행
• 부정적 표현이 줄어들거나 사라지면 정상 등록 처리

[그대로 등록] 클릭 시:
• 현재 내용 그대로 등록 처리
• 게시글 정상 등록됨
• 게시글에 해당 감정분석 라벨 표시
• 백엔드에서 해당 게시글을 모니터링 대상으로 자동 플래깅
• 시스템 관리자 대시보드에서 조회/관리 가능
```
````
