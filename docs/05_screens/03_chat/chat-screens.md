## 화면 목록

채팅 도메인의 각 화면은 개별 문서로 분리되어 있습니다.

| 화면 | 파일 | 경로 | 설명 |
|------|------|------|------|
| 채팅 목록 | [chat-list-page.md](./chat-list-page.md) | `/chat` | 채팅방 목록, 검색 |
| 채팅방 | [chat-room-page.md](./chat-room-page.md) | `/chat/[id]` | 채팅방, 메시지, 감정분석 |
| 채팅 만들기 | [chat-create-modal.md](./chat-create-modal.md) | - | 1:1 채팅, 그룹 채팅 생성 모달 |

---

## 화면 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  채팅 목록 (ChatListPage)                                                   │
│  ├── 채팅방 클릭 → 채팅방 (ChatRoomPage)                                   │
│  ├── 새 채팅 → 1:1 채팅 모달 → 채팅방                                     │
│  └── 그룹 만들기 → 그룹 채팅 모달 → 채팅방                                │
│                                                                             │
│  채팅방 (ChatRoomPage)                                                      │
│  ├── 메시지 전송 → AI 감정분석 → 경고 시 모달                             │
│  ├── 이미지 전송 → 라이트박스                                             │
│  └── 나가기 → 채팅 목록                                                   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 공통 규칙

### 메시지 정렬

| 메시지 유형 | 정렬 | 프로필/닉네임 |
|-------------|------|---------------|
| 본인 메시지 | 우측 정렬 | 표시 안함 |
| 상대방 메시지 | 좌측 정렬 | 중앙 라인 정렬 (프로필과 닉네임 세로 중앙 맞춤) |

### 프로필과 닉네임 정렬 규칙

```css
.message-row {
  display: flex;
  align-items: center;  /* 중앙 정렬 */
  gap: 8px;
}
```

- 프로필 사진과 닉네임은 메시지 버블의 **세로 중앙에 위치**
- 프로필 + 닉네임 + 버블이 하나의 수평 그룹으로 중앙 정렬

### AI 감정분석 결과 표시

| 라벨 | 아이콘 | 표시 위치 |
|------|--------|-----------|
| 긍정 | 😊 | 메시지 버블 하단 |
| 중립 | 😐 | 메시지 버블 하단 |
| 주의 | 😟 | 메시지 버블 하단 |
| 경고 | ⚠️ | 메시지 버블 하단 |

### 접근 권한

| 화면 | 권한 |
|------|------|
| 채팅 목록, 채팅방 | 로그인 필수 |
| 특정 채팅방 | 채팅방 참여자만 |
