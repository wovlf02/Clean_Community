# 친구 화면 설계 (Friends Screens)

**관련 문서**: [디자인 시스템](../00_common/design-system.md) | [기능 요구사항](../../02_requirements/functional.md)

---

## 화면 목록

친구 도메인의 각 화면은 개별 문서로 분리되어 있습니다.

| 화면 | 파일 | 경로 | 설명 |
|------|------|------|------|
| 친구 목록 | [friend-list-page.md](./friend-list-page.md) | `/friends` | 친구 목록, 온라인 상태, 친구 요청 |
| 친구 찾기 | [friend-search-page.md](./friend-search-page.md) | `/friends/search` | 친구 검색, 친구 요청 |

---

## 화면 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  친구 목록 (FriendListPage)                                                 │
│  ├── 전체 탭: 모든 친구 목록                                               │
│  ├── 온라인 탭: 현재 온라인인 친구만                                       │
│  ├── 친구 요청 탭: 받은/보낸 요청 관리                                     │
│  ├── 친구 찾기 → 친구 찾기 (FriendSearchPage)                              │
│  ├── 채팅하기 → 채팅방 (ChatRoomPage)                                      │
│  └── 친구 삭제 → 삭제 확인 모달                                            │
│                                                                             │
│  친구 찾기 (FriendSearchPage)                                               │
│  ├── 검색 → 검색 결과                                                      │
│  ├── 친구 요청 → 요청 완료                                                 │
│  └── 이미 친구 → 채팅하기                                                  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 공통 규칙

### 온라인 상태 표시

| 상태 | 아이콘 | 색상 |
|------|--------|------|
| 온라인 | ● | success-500 (초록) |
| 오프라인 | ○ | neutral-400 (회색) |

### 친구 요청 상태

| 상태 | 표시 | 가능한 액션 |
|------|------|-------------|
| 친구 아님 | - | [친구 요청] |
| 요청 대기 중 | 대기 중 | [요청 취소] |
| 이미 친구 | 이미 친구 | [💬 채팅] |
| 받은 요청 | - | [거절] [수락] |

### 접근 권한

| 화면 | 권한 |
|------|------|
| 친구 목록, 친구 찾기 | 로그인 필수 |
