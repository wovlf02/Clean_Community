---

## 화면 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  대시보드 (DashboardPage)                                                   │
│  ├── 통계 카드: 게시글, 댓글, 좋아요, 친구 수                              │
│  ├── 활동 통계 차트: 오늘/이번주/이번달 활동량                             │
│  ├── 인기 게시글: TOP 5 게시글 + 감정분석 결과                             │
│  ├── 최근 활동: 나의 최근 활동 기록                                        │
│  └── 알림: 최근 알림 목록                                                  │
│                                                                             │
│  네비게이션                                                                 │
│  ├── 인기 게시글 클릭 → 게시글 상세 (PostDetailPage)                       │
│  ├── 더보기 → 게시글 목록 (PostListPage)                                   │
│  └── 알림 클릭 → 해당 페이지로 이동                                        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
