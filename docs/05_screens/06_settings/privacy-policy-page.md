````markdown
## 3. 개인정보 처리방침 상세 내용

### 제1조 (개인정보의 처리 목적)

| 목적 | 상세 내용 |
|------|-----------|
| 회원 가입 및 관리 | 회원 가입의사 확인, 본인 식별·인증, 회원자격 유지·관리, 서비스 부정이용 방지, 각종 고지·통지 |
| 서비스 제공 | 게시판 서비스, 채팅 서비스, 친구 기능, AI 감정분석을 통한 건전한 커뮤니티 환경 조성 |
| 마케팅 (동의 시) | 신규 기능 안내, 이벤트 정보 제공, 서비스 개선을 위한 통계 분석 |

### 제2조 (개인정보의 수집 항목)

| 구분 | 수집 항목 |
|------|-----------|
| 필수 | 아이디, 비밀번호, 이메일, 닉네임, 이름 |
| 선택 | 프로필 이미지, 자기소개, 마케팅 수신 동의 |
| 자동 수집 | 기기정보, 방문기록, 쿠키, AI 감정분석 결과 |

### 제3조 (개인정보의 처리 및 보유 기간)

| 보유 항목 | 보유 기간 | 법적 근거 |
|-----------|-----------|-----------|
| 회원 정보 | 회원 탈퇴 시까지 (탈퇴 후 30일 유예) | - |
| 서비스 이용기록 | 3년 | 통신비밀보호법 |
| 접속로그 | 3개월 | 통신비밀보호법 |
| AI 감정분석 결과 | 게시물 삭제 시까지 | - |

### 제4조 (개인정보의 제3자 제공)

> 회사는 원칙적으로 정보주체의 개인정보를 제3자에게 제공하지 않습니다.
> 다만, 다음의 경우에는 예외로 합니다.
> - 정보주체가 사전에 동의한 경우
> - 법률에 특별한 규정이 있거나 법령상 의무를 준수하기 위한 경우
> - 수사 목적으로 법령에 정해진 절차와 방법에 따라 요청받은 경우

### 제5조 (개인정보의 처리 위탁)

| 수탁업체 | 위탁 업무 | 보유 기간 |
|----------|-----------|-----------|
| Amazon Web Services | 클라우드 서버 운영 | 위탁계약 종료 시 |
| (이메일 서비스 제공자) | 이메일 발송 | 발송 완료 시 |

### 제6조 (개인정보의 파기)

```
[파기 절차]
1. 보유기간 경과 또는 처리목적 달성 시 파기 계획 수립
2. 개인정보 보호책임자 승인
3. 개인정보 파기 실행

[파기 방법]
• 전자적 파일: 복원 불가능한 방법으로 영구 삭제
• 종이 문서: 분쇄기로 분쇄 또는 소각
```

### 제7조 (정보주체의 권리·의무 및 행사방법)

| 권리 | 행사 방법 |
|------|-----------|
| 열람 요구권 | 설정 > 개인정보 관리 또는 고객센터 |
| 정정·삭제 요구권 | 설정 > 프로필 관리 |
| 처리정지 요구권 | 고객센터 또는 개인정보 보호책임자 연락 |

### 제8조 (개인정보의 안전성 확보조치)

| 조치 사항 | 상세 내용 |
|-----------|-----------|
| 관리적 조치 | 내부 관리계획 수립·시행, 정기적 직원 교육 |
| 기술적 조치 | 개인정보처리시스템 접근 권한 관리, 암호화 기술 적용, 보안 프로그램 설치·운영 |
| 물리적 조치 | 전산실, 자료보관실 접근 통제 |

### 제9조 (개인정보 보호책임자)

```
````
