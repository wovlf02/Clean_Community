## 🚀 AWS 배포 빠른 체크리스트

### 빌드 전
- [ ] Mock 데이터 import 제거 완료
- [ ] 모든 API Routes에 `export const dynamic = 'force-dynamic'` 추가
- [ ] 환경변수 (.env.production) 설정 완료
- [ ] `npm run build` 에러 없음

### 빌드 후
- [ ] 게시글 목록이 API에서 로드됨
- [ ] 상대 시간 (3분 전 등)이 클라이언트에서 계산됨
- [ ] 채팅/알림 실시간 동기화 작동

### 배포 후
- [ ] OAuth 콜백 URL 설정 확인
- [ ] AI 서버, Socket 서버 연결 확인
- [ ] S3 이미지 로딩 확인
