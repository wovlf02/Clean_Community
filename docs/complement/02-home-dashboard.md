# 02. 홈 대시보드 및 랜딩페이지

## 현재 문제
1. 홈 버튼 클릭 시 자동으로 게시판으로 리다이렉트됨
2. 대시보드/랜딩페이지가 없음

## 개선 방안
1. `/` 경로에 대시보드 페이지가 정상 동작하도록 수정
2. 홈 네비게이션이 `/`로 올바르게 연결되도록 확인
3. 리다이렉트 로직 제거

## 구현 내용

### 파일 확인/수정
- `cc/src/app/(main)/page.tsx` - 대시보드 유지
- `cc/src/components/layout/header/Header.tsx` - 네비게이션 확인
- `cc/next.config.ts` - 리다이렉트 설정 확인
- 미들웨어 확인
