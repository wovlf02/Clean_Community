# 04. 채팅 레이아웃 개선

## 현재 문제
1. 채팅 메인 페이지에서 채팅방 리스트만 보임
2. 채팅방 클릭 시 페이지 전체가 채팅방 상세로 이동
3. 리스트와 상세 동시 확인 불가

## 개선 방안 (문서 기반)
1. 데스크톱: 좌측 채팅 리스트(320px) + 우측 채팅 상세(flex-1)
2. 채팅방 미선택 시 안내 메시지 표시
3. 채팅방 선택 시 우측에 상세 표시
4. 모바일: 기존 방식 유지 (페이지 전환)

## 구현 내용

### 파일 수정
- `cc/src/app/(main)/chat/page.tsx` - 스플릿 레이아웃
- `cc/src/app/(main)/chat/layout.tsx` (신규) - 채팅 레이아웃
- `cc/src/app/(main)/chat/chat-list.css` - 스타일 수정
- `cc/src/components/chat/chat-detail/` (신규) - 채팅 상세 컴포넌트
