# 05. 메시지 버블 스타일 수정

## 현재 문제
1. 상대방 메시지 전송 시각이 버블 좌측 하단에 위치
2. 내 메시지 전송 시각이 버블 우측 하단에 위치
3. 읽지 않은 참여자 수가 체크 표시로 구현됨 (문서와 다름)

## 설계 문서 기준
- **상대방 메시지**: 버블 우측 하단 바깥에 전송시각, 버블과 같은 하단 라인
- **내 메시지**: 버블 좌측 하단 바깥에 전송시각, 버블과 같은 하단 라인
- **읽지 않은 수**: 전송시각 바로 위에 숫자로 표시

## 구현 내용

### 파일 수정

- `cc/src/components/chat/message-bubble/message-bubble.css`

### 변경사항
```
[상대방 메시지]
┌───────────────────────┐
│  메시지 내용          │  3      ← 읽지 않은 수
│                       │  10:30  ← 전송 시각
└───────────────────────┘

[내 메시지]
      3  ┌───────────────────────┐
  10:31  │  메시지 내용          │
         └───────────────────────┘
```
