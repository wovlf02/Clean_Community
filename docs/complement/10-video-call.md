# 10. 화상 통화 UI

## 현재 문제
1. 화상 통화 기능 없음

## 개선 방안
1. 채팅방 상세 영역 전체를 화상통화 UI로 전환
2. 참여자별 모서리 둥근 직사각형 영상 표시
3. 캠/마이크 토글 기능
4. 참여자 리스트 (좌측 하단)
5. 통화 시간 (하단 중앙)
6. 통화 종료 버튼 (우측 하단)
