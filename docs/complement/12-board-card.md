# 12. 게시판 카드 개선

## 현재 문제
1. 썸네일 공간이 너무 많음 (16:9 비율)
2. 감정분석 점수가 표시되지 않음

## 개선 방안
1. 썸네일을 좌측 정사각형 또는 제거
2. 가로형 카드 레이아웃으로 변경
3. 감정분석 점수 뱃지 추가

## 구현 내용

### 새 카드 레이아웃
```
┌─────────────────────────────────────────────────────────────────┐
│ ┌──────┐                                                        │
│ │      │  [일반]  게시글 제목                          😊 +12   │
│ │ 썸네일│  게시글 내용 미리보기...                              │
│ │      │  👤 닉네임 · 3분 전          👁 123  ❤ 45  💬 12      │
│ └──────┘                                                        │
└─────────────────────────────────────────────────────────────────┘
```

### 감정분석 점수
- 긍정: 😊 +점수 (초록색)
- 중립: 😐 점수 (회색)
- 부정: 😟 -점수 (주황색)
- 경고: ⚠️ 점수 (빨간색)
