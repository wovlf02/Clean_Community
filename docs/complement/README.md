## 개선 사항 목록

이 문서는 플랫폼 UI/UX 개선 요청사항을 분석하고 구현 계획을 정리합니다.

### 1. [헤더 레이아웃 개선](./01-header-layout.md) ✅
- 감성 커뮤니티 타이틀 좌측 끝 정렬
- 네비게이션 좌측 배치
- 프로필/알림/검색 우측 끝 배치
- 플랫폼 아이콘 배치 공간 마련

### 2. [홈 대시보드 및 랜딩페이지](./02-home-dashboard.md) ✅
- 홈 버튼 → `/` 경로 정상 연결
- 대시보드 페이지 구현 완료

### 3. [헤더 우측 기능 구현](./03-header-actions.md) ✅
- 검색, 알림, 프로필 기능 구현
- 우측 끝 정렬

### 4. [채팅 레이아웃 개선](./04-chat-layout.md) ✅
- 데스크톱 스플릿 뷰 구현 (리스트 + 상세)
- 채팅방 실시간 알림

### 5. [메시지 버블 스타일 수정](./05-message-bubble.md) ✅
- **상대방 메시지**: 버블 좌측 상단에 닉네임 표시
- **상대방 메시지**: 버블 우측에 시간/읽지않은수
- **내 메시지**: 버블 좌측에 시간/읽지않은수
- 카카오톡 스타일 레이아웃

### 6. [메시지 정렬 수정](./06-message-alignment.md) ✅
- 프로필 사진 없는 메시지 정렬 유지 (placeholder)

### 7. [메시지 레이아웃 구조 변경](./07-message-layout.md) ✅
- 닉네임/프로필 사진 배치 수정

### 8. [읽지 않은 참여자 수 계산](./08-unread-count.md) ✅
- 본인 제외 로직 구현

### 9. [WebRTC 통화 기능](./09-webrtc-call.md) ✅
- 음성/화상 통화 구현
- 오버레이 UI

### 10. [화상 통화 UI](./10-video-call.md) ✅
- 참여자별 영상 표시
- 참여자 리스트

### 11. [채팅 미디어 기능](./11-chat-media.md) ✅
- 이모티콘 구현
- 파일/이미지 업로드
- **IME 입력 문제 해결** (한글 입력 시 중복 전송 방지)

### 12. [게시판 카드 개선](./12-board-card.md) ✅
- **그리드 레이아웃** 적용 (1/2/3열)
- 썸네일 최소화 (48px, 우측 상단)
- 제목, 내용, 작성자, 통계 정보 강조
- 감정분석 점수 표시

### 13. [친구 페이지 필터 CSS](./13-friend-filter-css.md) ✅
- 탭 스타일 완전 개선
- hover/active 상태 명확화

### 14. [친구 기능 개선](./14-friend-features.md) ✅
- 채팅 아이콘 기능 구현
- 친구 추가 모달
